# 42sh

The goal of this project was to reproduce a bash style shell.

## Getting Started

### What you need

- An Unix environement

### Prerequisites & How to get them

* There is no prerequisites, everything you need should be in the library folder.

### How to run it ?

- `make`
- `./42sh`
